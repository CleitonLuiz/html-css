Usando links Internos

Está é a primeira página do meu site, voçê também pode acessar segunda página

Para usar link interno preciso criar um doc dentro da mesma pasta do html, usa a tag a href para indicar o link, colocando dentro o rel next, para voltar rel prev, e marcar o texto que quero indicar para voltar.
